# 定时任务调度命令

## `crond`任务调度

> 系统后台有一个`crond`的程序，可以完成任务调度功能。系统在某个时间执行的特定命令或程序

### 任务调度分类

- 系统工作
  有些重要的工作必须周而复始的执行，例如病毒扫描

- 用户工作
  个别用户可能希望执行某些程序，例如对`MySQL`数据库的备份

### 任务调度命令

- `crontab`

## `atd`任务调度

> `atd`后台程序是`at`命令的守护进程，`atd`程序每隔60s检查作业队列，同时会执行时间匹配的上的定时任务

### 任务调度命令

- `at`
  一次性的执行匹配上的定时任务
- `atq`
  查看任务列表
- `atrm 任务编号`
  删除指定的任务编号任务
